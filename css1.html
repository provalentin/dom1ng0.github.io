Что такое CSS

Этот минисайт далек от идеала. И только CSS может нам помочь сделать его красивым.

CSS (Cascade Style Sheet) - это специальный язык, который фактически дополняет существующую разметку. Этот язык не является языком программирования и обрабатывается браузером.

Пример

Мы говорим ему: "выбери все ссылки в документе и примени к ним свойство uppercase".

			
a {
	text-transform: uppercase;
}

Преимущества

  • Содержание отделяется от оформления
  • Для разных типов устройств - свое оформление
  • Ускоряют обработку веб страниц
  • Один файл со стилями на множество документов

Как выглядит код CSS

Селектор - "фильтр", который выбирает элемент или группу элементов.

Свойство - значение.

selector {
  property-1 : value-1;
  property-2 : value-2;
  property-N : value-N;
}

Подключаем CSS к документу

  • Используем атрибут style="color: red;"
  • Используем тег <style>
  • Подключаем внешний файл: 
    
	<link rel="stylesheet" type="text/css" href="style.css">
  • Указываем тип устройства media="...": https://css-tricks.com/snippets/css/all-stylesheet-media-types/

Базовые селекторы

  • Любой элемент * { }
  • Селектор по названию тега span { }
  • Идентификаторы #element_id { }
  • Классы .element_class { }

Сложные селекторы

  • Комбинации span.element_class, p, #my_id { }
  • Вложение .foo .bar { }
  • Дочерний .foo > .bar { }
  • Селекторы атрибутов input[type=text], [data-var="..."] { }
  • Псевдоклассы a:hover { }

Приоритет
