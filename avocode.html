Tools

  • Рука - перемещение по макету
  • Стрелка - курсор; взаимодействие с элементами макета
    • Один клик - выделение елемента для дальнейшего
      взаимодействия, возможность посмотреть его свойства
    • Двойной клик - копирование текста / экспорт элемента
    • Выделение нескольких элементов
Сделать экспорт картинки из макета в формате .svg

  • Линейка - измерение расстояний между элементами
Измерить растояние между двумя элементами верхнего меню

  • Пипетка - определение цвета
    • Зажатие курсора позволяет узнать цвет элемента на пиксельном уровне
    • В верхнем правом меню находятся уже определенные цвета

В нижнем левом меню можно поменять формат представления цвета

Определить цвет любой точки фона в формате RGB

  • Нож - вырезание и любой области из макета
Вырезать и экспортировать лого макета
